# Fluento
# English Version
Fluento is a gamified language learning web application designed to make learning English more engaging, interactive, and accessible for learners of different ages. Inspired by modern platforms like Duolingo, Fluento focuses on combining education with simple game-like elements to create a fun and motivating learning environment.

The main objective of Fluento is to simplify the language learning process through a clean interface, structured learning paths, and interactive activities. Instead of presenting lessons in a traditional and sometimes overwhelming format, Fluento delivers content through short exercises, quizzes, and visual learning components that help users stay focused and motivated.

Fluento provides three main learning modes that serve different types of users. The first mode, Fluento Kids, is designed for younger learners. It features a colorful interface, beginner-friendly exercises, and simple learning materials that help children start learning English in a fun and approachable way.

The second mode, Fluento Knight, is aimed at teenagers and general learners who want to improve their English skills through a more structured learning experience. This mode includes lessons, quizzes, and challenges that gradually increase in difficulty, encouraging users to build vocabulary, reading comprehension, and language confidence over time.

The third mode, Fluento Translate, is a built-in translation tool that helps users quickly translate words or sentences. This feature supports the learning process by allowing users to instantly understand unfamiliar vocabulary while studying.

The project focuses on building a modern and responsive user interface that is easy to navigate and visually appealing. Fluento aims to demonstrate how gamification and simple design principles can improve the experience of learning a new language in a web environment.

Fluento is built using modern frontend technologies such as React, React Router, TailwindCSS, and JavaScript to create a fast, responsive, and interactive web experience.

# Fluento
# Versi Indonesia
Fluento adalah aplikasi pembelajaran bahasa berbasis web yang dirancang untuk membuat proses belajar bahasa Inggris menjadi lebih menarik, interaktif, dan mudah diakses oleh berbagai kalangan pengguna. Terinspirasi dari platform pembelajaran modern seperti Duolingo, Fluento menggabungkan konsep pendidikan dengan elemen gamifikasi sederhana untuk menciptakan pengalaman belajar yang menyenangkan dan memotivasi.

Tujuan utama Fluento adalah menyederhanakan proses belajar bahasa melalui tampilan antarmuka yang bersih, jalur pembelajaran yang terstruktur, serta aktivitas interaktif. Dibandingkan dengan metode pembelajaran tradisional yang sering terasa membosankan atau terlalu kompleks, Fluento menghadirkan materi melalui latihan singkat, kuis, dan komponen visual yang membantu pengguna tetap fokus dan termotivasi.

Fluento menyediakan tiga mode pembelajaran utama yang ditujukan untuk jenis pengguna yang berbeda. Mode pertama adalah Fluento Kids, yang dirancang khusus untuk anak-anak. Mode ini memiliki tampilan yang lebih berwarna, latihan yang sederhana, serta materi dasar yang membantu anak-anak mulai belajar bahasa Inggris dengan cara yang menyenangkan.

Mode kedua adalah Fluento Knight, yang ditujukan untuk remaja dan pengguna umum yang ingin meningkatkan kemampuan bahasa Inggris mereka melalui pengalaman belajar yang lebih terstruktur. Mode ini menyediakan berbagai materi pembelajaran, kuis, serta tantangan yang tingkat kesulitannya meningkat secara bertahap.

Mode ketiga adalah Fluento Translate, yaitu fitur penerjemah yang memungkinkan pengguna menerjemahkan kata atau kalimat dengan cepat. Fitur ini membantu proses belajar dengan memberikan pemahaman langsung terhadap kosakata yang belum dikenal.

Fluento dikembangkan dengan fokus pada desain antarmuka modern yang responsif, mudah digunakan, dan nyaman untuk dinavigasi. Proyek ini bertujuan untuk menunjukkan bagaimana konsep gamifikasi dan desain web yang sederhana dapat meningkatkan pengalaman belajar bahasa dalam lingkungan digital.

Fluento dibangun menggunakan teknologi modern seperti React, React Router, TailwindCSS, dan JavaScript untuk menghasilkan aplikasi web yang cepat, responsif, dan interaktif.
